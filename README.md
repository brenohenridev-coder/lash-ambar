## Como Usar

### Localmente
1. Abra o arquivo `index.html` no navegador
2. Ou use um servidor local: `npx serve` ou `python -m http.server`

### Personalização

#### Trocar imagens
Substitua as URLs no `src` das tags `<img>` por suas próprias fotos:
```html
<img src="sua-foto-aqui.jpg" alt="Descrição">
```

#### Alterar telefone WhatsApp
Em `index.html`, procure por `5511999999999` e substitua pelo seu número.

Em `script.js`, na linha 142, substitua o mesmo número.

#### Alterar cores
No arquivo `styles.css`, edite as variáveis CSS no `:root`:
```css
--color-accent: #c9a96e;      /* Cor principal (dourado) */
--color-primary: #1a1a1a;     /* Cor escura */
--color-off-white: #faf9f7;   /* Cor clara */
```

#### Alterar textos
Simplesmente edite o conteúdo no `index.html` mantendo as tags HTML.

### Otimizações Recomendadas

1. **Imagens**: Use WebP para melhor performance
2. **CDN**: Hospede em Vercel, Netlify ou Cloudflare Pages
3. **SEO**: Adicione meta tags específicas para sua região
4. **Analytics**: Integre Google Analytics ou Plausible
